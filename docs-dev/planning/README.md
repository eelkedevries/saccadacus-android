# planning

Planning material: roadmap notes, implementation notes, open questions, and `current_state.md` — a living, high-level orientation (systems present, key decisions, what's in progress) that the agent reads at session start and updates as the project evolves.

This folder records development decisions and intent. Reference specific files from prompts when background is needed. Do not include this folder in the deployed build output. If the repository is public this material is visible, so keep secrets and credentials out of it.
